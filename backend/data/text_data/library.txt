===TITLE: 대출 및 반납 안내
도서 대출 및 반납 절차:
- 대출 방법:
  1) 도서관 홈페이지 또는 검색전용PC에서 자료 검색
  2) 소장위치, 도서상태(대출가능), 청구기호 확인
  3) 서가에서 직접 자료 찾기
  4) 자료실(2,3층) 무인대출기 또는 대출데스크(1층)에서 학생증(스마트 카드) 인식 후 대출
  5) 무인대출기 이용 시 확인증 출력하여 반납예정일 확인
- 무인대출기 이용 불가 경우:
  * 대출 가능 권수를 초과한 경우
  * 연체 자료가 있는 경우
  * 딸림자료(CD-ROM, 책자 등)가 있는 경우
  * 학생증(모바일)이 없는 경우
- 반납 방법:
  * 반납대 또는 무인반납기에 반납
  * 딸림자료(CD-ROM, Tape 등) 반드시 함께 반납
  * 반납 후 처리 결과 반드시 확인
- 무인반납기 이용:
  * 위치: 1층 학생회관 쪽 출입구 방향 (1층 제증명발급기 옆)
  * 운영: 365일 24시간
  * 근무시간 외 또는 휴일 반납 가능
  * 무인반납기 반납 도서는 다음 근무일에 반납 처리
  * 딸림자료(CD, Tape 등)가 있는 도서, 예약도서는 무인반납기에서 반납 불가
- 유의사항: 학생증 재발급 시 대출대에 등록 필요
- 문의: 대출실 061-750-5020, 5021

===TITLE: 대출 자료 예약
대출 자료 예약 서비스:
- 예약 대상: 찾는 자료가 모두 '대출중' 상태인 경우
- 예약 가능 권수: 1인 2종
- 예약 대기자: 1책(동일도서) 2인까지 가능
- 예약 가능 대상: 교직원, 학부생, 대학원생
- 예약 방법:
  1) 도서관 홈페이지 접속
  2) 로그인
  3) 자료 검색
  4) 검색결과에서 "예약신청" 클릭 (동일도서 중 "대출가능" 상태의 도서가 없어야 함)
  5) 대출예약 완료
- 예약 조회 및 취소:
  * 도서관 홈페이지 → My Library → 대출예약 상황조회
  * 대출목록 아래 예약목록에서 개별 취소 가능
- 예약 도서 도착 안내:
  * SMS 또는 도서관 홈페이지 My Library → 개인공지사항으로 전달
- 예약 도서 대출:
  * 도착 안내 통보 후 2일 내에 대출
  * 2일 내 미대출 시 자동 예약 취소, 다음 예약자에게 대출 또는 자료실 이관
- 유의사항:
  * 대출중인 자료가 예약신청되면 대출연장 불가
  * 예약 신청한 자료를 대출하지 않아 예약 취소 시 이후 30일간 예약 제도 이용 불가
  * 웹상 반납예정일은 대출중인 이용자의 반납예정일이며 실제 반납일과 다를 수 있음
- 문의: 대출실 061-750-5020, 5021
